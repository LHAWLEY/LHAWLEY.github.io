Recursion

January 11, 2016

What is recursion? -- Recursion is the idea that a function or method can call itself, passing in a different argument each time until it reaches a certain threshold. Recursion is often used to replace loops in a method.

Recursion breaks down a problem into smaller pieces. In computer science this algorithm is known as divide and conquer.

When using recusion it is important to add a base case that tells your method when to terminate. If you do not add this element you run the risk of creating an infinate loop that can cause your program to crash.
