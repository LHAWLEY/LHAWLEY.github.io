Enumerable Methods

December 14, 2015

Enumerable#map takes an array runs it through a block of code and returns a new array.

Ex: (1..6).map {|i| i + 2} => [3, 4, 5, 6, 7, 8]

Enumerable#group_by is a method that takes an array and returns a hash. With the keys as the evaluated results of from the block and the values being the arrays of elements that correspond to the key.

Ex: (1..6).group_by {|i| i + 2} => {[0] 3, [1] 4, [2] 5, [3] 6, [4] 7, [5] 8]

Enumerable#cycle is a method that call on a block for each elemenr for a set number of times or for ever.

Ex: (1..6).cycle(2) {|x| puts x} => prints 1,2,3,4,5,6,1,2,3,4,5,6
