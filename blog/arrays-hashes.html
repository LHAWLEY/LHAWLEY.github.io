Arrays and Hashes

December 7, 2015

Arrays and hashes are two different data structures in Ruby. Arrays are structured in a list between [ ].

Array's can contain numbers [1, 2, 3, 4]; strings ["this", "is", "an, "array"]; other arrays [[1, 2, 3],[4, 5, 6],[7, 8, 9]] ;and they can also be empty [ ].

Each array element has an index number starting a [0] so that you easily referance a specific element or an array. An example is: favorite_number [12, 6, 99, 24] If I wanted to retrieve 6 I would say: favorite_number[1]

A hash is like an array execept insead of having values that are associated with an index. Hashes have key value pairs.

Here is an example of a hash my_pets = { my_dog: "Sophia" my_cat: "Fluffy" my_bird: "Tweets" }

Like an array you can retrieve the values of a hash. If I wanted to retrieve my dog I would say my_pets[my_dog] to return the value that is associated with that key which would be "Sophia".
