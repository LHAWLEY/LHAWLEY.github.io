What is in a Class?

December 20, 2015

Some definitions to keep in mind as we move through this blog post:

Class: A class is an object much like everything in Ruby. However, a class has the ability to create new objects or instances of the class object.

Example Class: 

          class Boat

          end

Instance Variable: An instance variable is a variable that is defined in a class. An instance variable starts with an @ symbol. Instance variables can be used to share information between in methods within a specific class. This is different from a local variable which is a variable that is housed inside of a method and can only be used in that method.

Example Instance Variable: 

          @color

Instance Method: An instance method is a method that is created within a class. Instance methods can be used on instances of a class only.

Example Instance Method: 
      def horn
        return  "toot, toot"
      end

Why do we create a class?

You want to create a class when you want methods to work on specific objects. By creating a class you can add specific methods to that class and those methods can be used on only those class objects.

What goes into a class?

Classes contain instance variables, instance methods and class variables.

Example of a Boat class 


      class Boat

      attr_accessor :color, :length, :sails

        def initialize(color, length, sails)
          @color = color
          @length = length
          @sails = sails
        end

        def sailboat
          return 'My boat is #{@color}, #{@length} feet with #{@sails} sail(s)'
        end

        def horn
          return "toot, toot"
        end
      end

      boat = Boat.new('white', 50, 2)
      p boat.sailboat => 'My boat is white, 50 feet with 2 sail(s)'
      p boat.horn  => 'toot, toot'

Within the Boat class we can give our boat a color a length and a number of sails we can also give our boat a horn.
