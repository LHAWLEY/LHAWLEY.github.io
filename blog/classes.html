What is in a Class?

December 20, 2015

Some definitions to keep in mind as we move through this blog: Class: A Class is an object much like everything in Ruby. However, a class has the ability to create new objects or instances of the class object. Example Class: class Dog end Instance Variable: An instance variable is a variable that is defined in a class. An instance variable starts with an @ symbol. Instance variables can be used in methods with in a specific class. This is different from a local variable which is a variable that is housed inside of a method and can only be used in that methond. An instance variable is a variable that is defined in a class. Example Instance Variable: @name = name Instance Method: Example Instance Method: def name @name end

Why do we create a class? You want to create a class when you want methods to work on specific objects. By creating a class you can add specific methods to that class and those methods can be used on only those class objects.

What goes into a class? Classes contain instance variables, instance methods and classe variables.

Example of a Class class Dog def initialize(name) @name = name end def name @name end def bark "woof" end end
